# CyberShield
Projet étudiant 
CyberShield
Plateforme Anti-Cybercriminalité Locale et de Dépôt de Preuves Numériques, Cette plateforme citoyenne et policière est dédiée à la lutte contre les cyber-escroqueries locales (phishing, arnaques aux faux investissements, escroqueries sur les réseaux sociaux, Mobile Money fraudes).
Description
CyberShield permet aux citoyens de signaler des arnaques en ligne et aux enquêteurs de la brigade de cybercriminalité de traiter les dossiers.

Technologies
Interface : HTML5, CSS3, JavaScript
Backend : Node.js, Express
Base de données : PostgreSQL
Installation
cd backend
npm install
node server.js
